Installation
--------

.. code::

   pip install git+https://github.com/biosustain/maudtools.git#egg=maudtools


Features
--------

Add multivariate formation energy priors from equilibrator to a maud input folder

.. code::

   maud_tools fetch-dgf-priors path/to/maud_input
